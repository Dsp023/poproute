## üéØ Learning Paths by Goal

### Goal: Get a Job in ML
1. Complete Andrew Ng's ML course
2. Learn Python + libraries (NumPy, Pandas, scikit-learn)
3. Complete 3-5 Kaggle competitions
4. Build portfolio projects
5. Study algorithms & data structures
6. Practice system design for ML

### Goal: Build LLM Applications
1. Learn Python basics
2. Complete Hugging Face NLP course
3. Study LangChain documentation
4. Build simple RAG application
5. Deploy to production
6. Learn prompt engineering

### Goal: Research in AI
1. Strong math foundation (Linear Algebra, Calculus, Probability)
2. Read foundational papers
3. Implement papers from scratch
4. Contribute to open-source projects
5. Pursue graduate degree
6. Publish papers
